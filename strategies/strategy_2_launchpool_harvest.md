# Стратегия 2: Launchpool Harvest Short — Фармеры на выход

> **Стартовый капитал:** $200 | **Плечо:** 3x | **Горизонт сделки:** 24–48 часов
> **Биржи:** Bybit Pre-Market, OKX Pre-Market | **Тип:** проактивный шорт перед структурным дампом

---

## Философия

Binance и Bybit регулярно запускают Launchpool: пользователи стейкают BNB/USDT несколько дней и получают токены нового проекта **бесплатно**. В день листинга они продают. Это не паника и не случайность — это **структурный, предсказуемый дамп с известной датой**.

Ты знаешь три вещи заранее:
1. Точную дату листинга (объявляется за 4–7 дней)
2. Объём токенов, которые будут выброшены на рынок
3. Исторический паттерн поведения фармеров

Средняя себестоимость токенов у фармеров = **ноль**. Стимул продать в первый день — стопроцентный.

---

## Чем отличается от Token Splash

| | Token Splash | Launchpool Harvest |
|---|---|---|
| Сигнал | RSI > 85 + Volume Spike после листинга | Известен за 4–7 дней до листинга |
| Момент входа | Реактивный (после пампа) | Проактивный (до дампа) |
| Конкуренция | Все видят RSI | Никто не мешает открыть шорт заранее |
| Точность тайминга | Средняя | Высокая (дата листинга фиксирована) |
| Стресс | Высокий | Низкий |

## Фильтры отбора

Обязательные:
```
Сигнал: Binance/Bybit/OKX объявляют Launchpool, HODLer Airdrop или Megadrop
Фильтр 1: FDV на листинге > $300M (хайп-токены дампят сильнее)
Фильтр 2: Float < 25% от total supply (мало токенов = легко двигать цену)
Фильтр 3: Farming rewards разблокированы 100% в день листинга (без вестинга)
Фильтр 4: Есть Pre-Market фьючерс на Bybit или OKX
```

Дополнительные (повышают уверенность):
```
+ Нет крупного маркетмейкера с известным именем
+ Период фарминга ≤ 7 дней (быстрый фарм = больше нетерпения продать)
+ Нет реального продукта в production (только roadmap)
```

---

## Логика входа

```
1. Launchpool объявлен → проверяешь фильтры
2. За 12–24 часа до официального листинга:
   a. Смотришь Pre-Market цену на Bybit/OKX
   b. Если Pre-Market цена на 20–40% выше справедливой оценки → входишь в шорт
3. Параметры позиции:
   Маржа: 20% от текущего капитала
   Плечо: 3x
   Стоп-лосс: +12% от входа (цена ушла выше — нарратив сильнее дампа)
   TP1: −25% от входа → закрыть 50% позиции
   TP2: −45% от входа → закрыть остаток
   Тайм-стоп: 48 часов (закрыть всё без исключений)
```

## Параметры ордера

| Параметр | Значение |
|---|---|
| Плечо | **3x** |
| Размер маржи | **20% капитала** |
| Стоп-лосс | **+12% от входа** |
| TP1 (50% позиции) | **−25% от входа** |
| TP2 (50% позиции) | **−45% от входа** |
| Тайм-стоп | **48 часов** |

**Мультипликаторы прибыли на маржу при 3x:**
- TP1 (−25% × 3x × 50% позиции) = **+37.5%** маржи
- TP2 (−45% × 3x × 50% позиции) = **+67.5%** маржи
- **Full Win** (TP1 + TP2) = **+105%** маржи
- **Partial Win** (только TP1) = **+37.5%** маржи
- **Stop Loss** (−12% × 3x × маржа) = **−36%** маржи

## Что строить в Go

```go
// Псевдокод логики бота
1. Мониторинг анонсов:
   - https://announcements.binance.com (Launchpool, HODLer, Megadrop)
   - https://announcements.bybit.com (Launchpool)
   - RSS/HTML парсинг каждые 5 минут

2. Детект ключевых слов: "Launchpool", "HODLer Airdrop",
   "farming period", "listing date"

3. Парсинг параметров:
   - listing_date, total_supply, farming_amount, fdv_estimate

4. Расчёт метрик:
   float_pct = farming_amount / total_supply
   fdv_ok = fdv_estimate > 300_000_000

5. Фильтрация:
   if float_pct < 0.25 && fdv_ok && no_vesting:
       schedule_trade(listing_date - 12h)

6. За 12ч до листинга:
   - Открыть шорт через Pre-Market API Bybit/OKX
   - Установить SL (+12%), TP1 (−25%, 50%), TP2 (−45%, 50%)
   - Тайм-стоп: 48h

7. Мониторинг позиции и логирование P&L
```

### Ключевые API Bybit Pre-Market
```
POST /v5/order/create
  category: "linear"
  symbol: "{TOKEN}USDT"
  orderType: "Market"
  side: "Sell"      # шорт
  qty: "{size}"
  positionIdx: 0    # one-way mode
```

## Антипаттерны — когда не входить

```
НЕ входить, если:
[ ] Маркетмейкер известен и активен (Wintermute, GSR)
    → обычно указывается в объявлении листинга
[ ] Pre-Market цена уже упала >20% за 24 часа перед листингом
    → дамп начался раньше, лучшая точка уже позади
[ ] FDV проекта < $300M
    → не проходит Фильтр 1 (порог совпадает с обязательным условием отбора)
[ ] Позитивный нарратив в топе крипто-СМИ за 48h
    → покупатели перевешивают фармеров
[ ] Float > 50%
    → половина саплая уже в руках у долгосрочных холдеров, давление меньше
[ ] Нет Pre-Market фьючерса на Bybit или OKX
    → невозможно войти проактивно (Фильтр 4)
```
