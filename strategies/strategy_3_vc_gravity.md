# Стратегия 3: VC-гравитация — High FDV / Low Float Structural Short

> **Стартовый капитал:** $200 | **Плечо:** 2x | **Горизонт сделки:** 2–8 недель
> **Биржи:** Bybit, OKX, Binance | **Тип:** структурный средне-срочный шорт

---

## Философия

Это системная уязвимость в токеномике, которую рынок воспроизводит год за годом.

VC-фонды и команды получают токены по цене раунда (например, $0.10 за токен). На листинге токен торгуется по $1.50 при FDV $15B и float 8% — то есть **92% токенов ещё не в обращении**. Это искусственный дефицит.

Дальше происходит предсказуемое: каждый месяц очередной вестинг разблокирует токены VC, те продают по рынку. Float медленно растёт с 8% до 15%, 25%, 40%... и каждый шаг вниз по цене — **математически неизбежен**.

Это не спринт (как Token Splash). Это марафон. Ты открываешь позицию сразу после листинга и держишь 4–8 недель.

## Фильтр входа

### Ключевой показатель: FDV/MCap Ratio
```
FDV_ratio = FDV_at_listing / Initial_Market_Cap
Если FDV_ratio > 8x → потенциальная цель
```

### Дополнительные фильтры
```
✓ Крупные VC-инвесторы с долей >20% total supply (a16z, Sequoia, Paradigm)
✓ Первый вестинг-анлок через 6–12 месяцев (cliff) — после него давление нарастает
✓ Нет реального продукта в production (только roadmap и нарратив)
✓ Категория: L2, bridge, DeFi-инфра (не мемы — поведение непредсказуемо)
✓ Текущий float < 20%
```

### Источники данных
- **CoinMarketCap** → вкладка "Vesting" / "Tokenomics"
- **tokenunlocks.com** — самый детальный календарь разлоков
- **CryptoRank** — FDV, float, данные о раундах финансирования

## Параметры позиции

| Параметр | Значение | Логика |
|---|---|---|
| Плечо | **2x** | Долгосрочный шорт — меньше плечо, меньше ликвидации |
| Размер маржи | **20% капитала** | Одна позиция, держим несколько недель |
| TP1 (50% позиции) | **−30% от входа** | Зафиксировать половину при первом значимом движении |
| TP2 (50% позиции) | **−50% от входа** | Остаток добираем на продолжении |
| Стоп-лосс | **+10% от входа** | Жёсткий, без исключений |
| Тайм-стоп | **60 дней** | Если цель не достигнута за 2 месяца — выход |

**Мультипликаторы прибыли на маржу при 2x:**
- TP1 (−30% × 2x × 50% позиции) = **+30%** маржи
- TP2 (−50% × 2x × 50% позиции) = **+50%** маржи
- **Full Win** (TP1 + TP2) = **+80%** маржи
- **Partial Win** (только TP1) = **+30%** маржи
- **Stop Loss** (+10% × 2x) = **−20%** маржи

## Ключевые риски стратегии

### Риск 1: Позитивный катализатор
ZRO в феврале 2026: LayerZero анонсировал новый L1 "Zero" — цена выросла с $1.63 до $2.50 (+53%) за несколько дней. Если позиция была открыта по $4.20 и TP1/$2.94 уже взят, это некритично. Но если только входил — стоп на $4.62 (+10%) был бы вышиблен.

**Правило:** Следи за новостями о проекте. Мажорный анонс (новый L1, партнёрство с национальным банком) — повод закрыть позицию досрочно.

### Риск 2: Бычий рынок меняет скорость
В бычий рынок токены не дампят сразу — они просто растут медленнее рынка. Дамп наступает, но позже. Тайм-стоп в 60 дней может срабатывать в убыток в сильный бычий рынок.

**Правило:** Открывай позиции после первого значимого отката от ATH (токен уже не растёт), а не сразу при листинге.

### Риск 3: Funding Rate
При 2x плече и 60-дневном удержании: если все шортят (high short interest), фандинг уходит в минус.
- При $400 нотионале: ~−$0.5 в сутки = −$30 за 60 дней
- Это 15% от маржи $200 — существенно

**Правило:** При funding rate > 0.1% в 8 часов — пересматривай позицию.

## Что строить в Go

```go
// Псевдокод трекера VC-токенов

type VCToken struct {
    Symbol       string
    FDV          float64
    CircMCap     float64
    FDVRatio     float64  // = FDV / CircMCap
    Float        float64  // = CircMCap / TotalMCap
    NextUnlock   time.Time
    UnlockAmount float64  // % total supply
}

func isCandidate(t VCToken) bool {
    return t.FDVRatio > 8.0 &&
           t.Float < 0.20 &&
           t.NextUnlock.Before(time.Now().AddDate(0, 3, 0))
}

// 1. Ежедневный парсинг новых листингов
//    API: CoinMarketCap, CryptoRank
// 2. Расчёт FDV Ratio для каждого нового токена
// 3. Если FDV Ratio > 8x → алерт в Telegram
// 4. Трекинг unlock schedule:
//    GET https://api.cryptorank.io/v1/currencies/{id}/vesting
// 5. Trailing stop: при падении −40% передвинуть стоп в безубыток
// 6. Принудительный выход через 60 дней
```
